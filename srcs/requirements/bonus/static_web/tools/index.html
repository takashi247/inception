Inception: Review instruction for bonus part

Redis cache

  • Jump to Wordpress admin page
  • Log in as admin account (Username: wordpress, Password: wordpress)
  • Go to "Setting" >> "Redis"

FTP server

  • Install lftp ("sudo apt-get update && sudo apt-get install lftp")
  • Connect to ftp server ("lftp -e 'set ssl:verify-certificate false' -u ftpuser -p 21 tnishina.42.fr")
  • Type in ftp password ("Password: ftppass")
  • Create text.txt in local directory ("!echo "aaaaa" > test.txt")
  • Upload text.txt to ftp server ("put test.txt")
  • Remove text.txt from local directory ("!rm test.txt")
  • Download text.txt from ftp server ("get test.txt")

Adminer

  • Jump to Adminer website
  • Log in as mariadb account (Server: mariadb, Username: wpuser, Password: dbpassword, Database: wpdb)
  • Check tables if posts and comments are properly stored

Additional: Grafana

  • Jump to Grafana website
  • Log in as grafana admin account (Username: gf_admin, Password: gf_admin_pass)
  • Click "Search" on the left panel and then go to "Wordpress statistics dashboard"
